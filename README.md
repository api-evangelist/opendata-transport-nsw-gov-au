# Transport for NSW Open Data Hub (opendata-transport-nsw-gov-au)

Transport for NSW Open Data Hub is a open data portal open-data portal for Australia running **CKAN**, exposing a standardized catalog API over ~230 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/opendata-transport-nsw-gov-au/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=opendata-transport-nsw-gov-au-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.3

## API
- **Transport for NSW Open Data Hub CKAN Action API** — base `https://opendata.transport.nsw.gov.au/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://opendata.transport.nsw.gov.au)
- Live endpoint verified: `https://opendata.transport.nsw.gov.au/api/3/action/status_show` · ~230 datasets
